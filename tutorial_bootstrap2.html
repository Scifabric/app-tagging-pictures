Tagging pictures tutorial

Hi! This is a demo application that shows how you can tag parts of pictures using the PyBossa framework in Crowdcrafting.org.

The application is really simple. It loads a photo from Flickr and asks you this question: Tag all the faces that you know

.

You then hover the mouse over tha faces that you know, and draw a box clicking in the image, and you will be able to type the name of the person and save it.

You can do it as many times as you want. Simple, right? Now let's try it!
